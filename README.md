# pcb-keychain

I built a car PCB keychain with a light up headlight which lights when you touch the wheel. I'm really proud of it and I spent a lot of time experimenting with different silkscreen and edge.cuts. I made it in easyEDA, because I wasnted to order from lcsc and I spent a rlly long time just learning how to use easyEDA. I built it because I made a blinky board before and loved it, so I wanted to make something that looked better and was cooler. I also made it very big to try and exploit jlcpcbs 4 dollar engineering fee.

more info can be found on my macando project:
